# 대시보드 필터/새로고침/모바일 개선 설계

## 목표
- 대시보드에서 그룹별로 포트폴리오를 좁혀 볼 수 있게 한다.
- 평가금액 전일대비 변화를 요약 지표로 표시한다.
- 대시보드와 종목 상세 화면에 명시적인 새로고침 UI와 마지막 갱신 시각을 제공한다.
- 모바일에서 카드, 필터, 표가 깨지지 않고 핵심 정보가 먼저 보이도록 정리한다.
- 손익 계산 기준을 명확히 분리한다.

## 계산 기준
- `투자원금`: 입금/출금 기준 순투입원금.
- `잔여원금`: 현재 남아 있는 buy lot의 원가 합계.
- `평가금액`: 현재가 기준 보유 평가금액.
- `평가손익`: `평가금액 - 잔여원금`.
- `총손익`: `평가금액 - 투자원금`.
- `전일대비 평가금액`: 현재 평가금액에서 같은 범위의 가장 최근 과거 history 평가금액을 뺀 값. 과거 값이 없거나 환산이 불가하면 `null`로 내려서 UI에서 `—`로 표시한다.

## 화면 설계
- 대시보드 상단에 그룹 필터를 추가한다. 선택지는 `전체`, 출처 그룹, 통합 그룹, `미분류`이다.
- 필터 선택 시 요약 카드, 차트, 보유종목은 선택된 그룹 기준 데이터를 사용한다. 그룹별 수익현황 표는 전체 비교용으로 유지한다.
- 대시보드와 종목 상세 상단에는 `새로고침` 버튼과 `마지막 갱신` 텍스트를 둔다.
- 모바일에서는 상단 컨트롤을 세로로 쌓고, 요약 카드는 작은 화면에서 1~2열, 표는 가로 스크롤로 유지한다.

## 검증
- 백엔드: 대시보드 응답이 평가손익/총손익/전일대비 평가금액을 올바르게 계산하는 테스트를 추가한다.
- 프론트: 그룹 필터, 새로고침 버튼, 손익 라벨, 모바일 친화 클래스가 렌더되는 테스트를 추가한다.
- 브라우저: Browser plugin으로 로컬 대시보드를 모바일 폭에서 확인한다.
