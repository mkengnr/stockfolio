# 공유 페이지 대시보드 개선 설계

## 목표

그룹 공유 페이지에도 메인 대시보드에서 제공하는 그룹 필터, 범위별 보유종목, 통합 차트를 적용한다.

## 공개 범위

- 출처 그룹 공유: 공유된 출처 그룹만 전체 범위로 표시한다.
- 통합 그룹 공유: 공유된 통합 그룹을 전체 범위로 표시하고 구성 출처 그룹을 필터 옵션과 누적 막대에 포함한다.
- 라벨 공유: 공유된 라벨만 전체 범위로 표시한다.
- 공유 범위 밖 그룹과 종목은 응답에 포함하지 않는다.
- DB 그룹 UUID, 보유종목 UUID, 공유 토큰은 공개 응답에 포함하지 않는다.

## 데이터 계약

공유 그룹 응답에 `dashboard`를 추가한다. 공개 대시보드는 다음 데이터를 제공한다.

- 공유 범위 전체 요약
- 통합 그룹 공유 시 구성 출처 그룹별 요약과 범위별 보유종목
- 공유 범위 전체 및 구성 출처 그룹의 날짜별 이력
- 공개 보유종목 행

그룹과 이력 연결에는 응답 내에서만 의미가 있는 `group_key`를 사용한다. 출처 그룹 UUID는 노출하지 않는다.

## UI

공유 페이지는 메인 대시보드와 같은 동작을 제공한다.

- 그룹 필터로 전체 또는 통합 그룹 구성 출처 그룹을 선택한다.
- 요약 카드는 선택 범위 값으로 변경한다.
- 보유종목 표는 선택 범위의 실제 잔여 lot 수량과 금액을 표시한다.
- 전체 선택 시 상단 차트에 구성 그룹별 누적 평가금액 막대, 전체 평가금액 실선, 투자원금 점선을 표시한다.
- 하단에는 일별 손익 증감 막대를 표시한다.
- 구성 그룹 선택 시 해당 그룹의 평가금액, 투자원금, 일별 손익만 표시한다.
- 기존 공유용 통화별 선 차트는 제거한다.

## 오류 및 보안

- 공개 스키마 변환 단계에서 내부 식별자를 제거한다.
- 로그인 필요 공유 링크의 기존 인증 게이트를 유지한다.
- 사용할 수 없는 값은 기존 대시보드와 동일하게 제외하고 경고는 내부 UUID를 마스킹한다.

## 테스트

- 통합 그룹 공유 응답에 구성 출처 그룹만 포함되는지 검증한다.
- 공개 응답에 그룹 UUID와 보유종목 UUID가 없는지 검증한다.
- 공유 페이지 그룹 필터가 요약, 차트, 보유종목을 함께 전환하는지 검증한다.
- 통합 차트가 공유 이력으로 누적 막대와 하단 손익 막대를 생성하는지 검증한다.
- 실제 운영 공유 URL에서 전체 및 구성 그룹 화면을 시각 검증한다.
